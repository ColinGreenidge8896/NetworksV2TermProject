Welcome To Our Term Project! We are Group Robot_1, Colin and Alex.  

In this project, we have a Visual Studio Project with our packet.h, MySocket.h and unit tests for both. 

We also have our Docker Shared Volume containing the packet.h, MySocket.h, and the CMakeLists.txt required to build the project in Linux. Also contained is the index.html and main.cpp required to run the webserver and GUI for users.

We are happy to say that as of now, the GUI fully communicates with the robot simulator, and is able to establish and communicate via UDP and TCP, send Drive, Status and Sleep commands, and request and retrieve telemetry responses. Packets are counted properly, and HTTP codes return correct values based on requests and responses. 

We are aware that some of our tests do not pass at this time, mostly because of changes we made to the prior Milestones over the course of the project. We have added comments as to why they don't pass, and want to fix them for the future. 

Thanks for your time.
